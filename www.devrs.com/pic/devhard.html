2000-03-02 - Mike Butler Linux PIC programmer - Posted by: Jeff Frohwein
Mike Butler wrote this Linux-based PIC programmer software in C++ (called picptk) and presently requires itcl3.0. Picptk supports the whole PIC family including all memory types (EEPROM, EPROM, and OTP). At present I've only populated it with a few family members (F84, C76, C50x) but adding new ones is simple and requires no code changes. (The code is all table driven.) Picptk also supports both serial and parallel programmers.
